How to import compiled Java (.class files) into Clojure.

Files: "run.bat", "jars/clojure-1.6.0.jar", "src/External.java", "main.clj"


jars/clojure-1.6.0.jar contains the clojure language.


run.bat sets classpath to jars/* (any files it finds in jar directory) and src/ directory

@ECHO OFF CALL java -cp jars/*;src/ clojure.main -i main.clj

src/External.java

public class External { public static int myFunction(int x) { return x*3; } // This function returns tripled any integer value passed to it }

main.clj

(println (External/myFunction 10))

Console: I manually compile External.java (It's on purpose that I don't automate this process)

project root>> cd src project root/src>> javac External.java project root/src>> cd.. project root>> _

Console: Then when everything is compiled and ready to go, I can run the project.

project root>> run 30 project root>> _
