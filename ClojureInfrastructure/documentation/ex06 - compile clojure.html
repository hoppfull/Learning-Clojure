How to compile Clojure files.

Files: "run.bat", "repl.bat" "clojure-1.6.0.jar", "main.clj"


clojure-1.6.0.jar contains the clojure language.


run.bat sets classpath to clojure-1.6.0.jar, current directory, and executes Main.class.

@ECHO OFF CALL java -cp clojure-1.6.0.jar;. Main

repl.bat starts the REPL

@ECHO OFF CALL java -cp clojure-1.6.0.jar;. clojure.main -r

main.clj is the starting point of a java application.

(ns Main (:gen-class)) (defn -main [& args] ; Not sure about the dash "-" prefix... (println "lulzc!"))

Console: Now we call the REPL to compile main.clj

project root>> run Error: Could not find or load main class Main project root>> repl Clojure 1.6.0 user => (set! *compile-path* "./") "./" user => (compile 'Main) Main user => (System/exit 0) project root>> run lulzc! project root>> _
