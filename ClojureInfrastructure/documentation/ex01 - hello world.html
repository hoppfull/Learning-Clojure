How to run clojure scripts.

Files: "run.bat", "clojure-1.6.0.jar", "main.clj"


clojure-1.6.0.jar contains the clojure language.


run.bat starts our main clojure script

@ECHO OFF CALL -java -cp clojure-1.6.0.jar;. clojure.main main.clj

main.clj contains our hello world script

(println "Hello World!")
