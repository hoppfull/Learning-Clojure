How to import Java files from another Java file.

Files: "compile.bat", "run.bat", "Main.java", "External.java"


compile.bat starts the java compiler. (You need the JDK with appropriate environment variables setup for this to work!) This will automatically compile Main.java and any other .java files that Main.java depends on into .class files.

@ECHO OFF CALL javac -cp . Main.java

run.bat calls the java runtime (You need a Java Runtime Environment for this to work!) and executes Main.class.

@ECHO OFF CALL java Main

External.java is our example of a module.

public class External { public static void printMessage(String s) { System.out.println(s); System.out.println(s); System.out.println(s); System.out.println(s); } }

Main.java is our application starting point. This calls our External.java module.

public class Main { public static void main(String args[]) { External.printMessage("Hello!"); } }
