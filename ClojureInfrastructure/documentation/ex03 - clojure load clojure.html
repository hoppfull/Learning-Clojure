How to load clojure scripts into another script.

Files: "run.bat", "clojure-1.6.0.jar", "main.clj", "external.clj"


clojure-1.6.0.jar contains the clojure language.


run.bat starts the clojure script main.clj. Command prompt cmd: "run"

@ECHO OFF java -cp clojure-1.6.0.jar;. clojure.main main.clj

external.clj is a module with a function definition we want to load into our main script.

(defn myFunc [s] (println s) (println s) (println s))

main.clj is our main script:

(load "external") (myFunc "yay external modules!")

Console:

project root>> run yay external modules! yay external modules! yay external modules! project root>> _
