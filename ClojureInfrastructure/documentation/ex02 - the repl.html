How to run the Clojure REPL.

Files: "repl.bat", "clojure-1.6.0.jar"


clojure-1.6.0.jar contains the clojure language.


repl.bat starts the clojure REPL

@ECHO OFF CALL java -cp clojure-1.6.0.jar clojure.main -r

Console: To start the REPL

project root>> repl Clojure 1.6.0 user=> _

Console: To close the REPL

user=> (System/exit 0) project root>> _
